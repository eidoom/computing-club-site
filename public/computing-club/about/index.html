About

2019-10-21 :: Ryan Moodie

Welcome to the online manifestation of the IPPP Computing Club! Here you’ll find information about upcoming sessions and write-ups of previous sessions.

Mailing list

To join the mailing list, please send a message to majordomo@durham.ac.uk with the following text in the message body: subscribe ph-complunchclub, which you can do by clicking here. You then have to follow the authorisation instructions which will be emailed back to you. Ensure all emails you send have a plain text body; authorisation will not work with HTML emails.

Ryan’s in charge of the mailing list; speak to him if there are any problems.

Website

Ryan’s in charge of the website; speak to him if you would like write permission. Alternatively, fork and submit a pull request.

There’s a guide for adding posts.

Meetings

Usually 1300 Mondays, OC304 or OC103.
